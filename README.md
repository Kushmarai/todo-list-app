To-Do List application demonstrating DOM manipulation and localStorage implementation.

# To-Do App
- A simple and clean To-Do List application built using HTML, CSS, and JavaScript.
- I created this project to practice DOM manipulation, events, and localStorage handling.

# Features:
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Stores tasks permanently using localStorage
- Simple, clean and responsive UI

# How It Works:
- When a task is added, it gets saved in browser localStorage
- Deleted tasks are also removed from storage
- Even after refreshing the page or closing the browser, your tasks remain saved
